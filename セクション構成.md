# 「可視化経営」実践会 LP セクション構成

## セクション一覧

1. **ヘッダーセクション** - ロゴ + CTAボタン（固定ヘッダー）
2. **ヒーローヘッダーセクション** - メインキャッチ + サブキャッチ + CTA
3. **問題提起・共感セクション** - 社長と社員の「ズレ」を可視化
4. **比較セクション** - 一般的な研修 vs 可視化経営実践会
5. **3つの成果物セクション** - 実践会で得られる3つの道具
6. **お客様の声・導入事例セクション** - テスティモニアル3件 + 実績数値 ※仮の内容
7. **こんな方におすすめ／向いていない方セクション** - ターゲット明確化
8. **講師プロフィール・ストーリーセクション** - 吉田多絵のプロフィール + 書籍紹介
9. **サービスの流れ・価格セクション** - 4ステップのタイムライン + 料金表
10. **リスク喚起・FAQセクション** - 注意事項 + よくある質問（アコーディオン）
11. **緊急性・限定性セクション** - 毎月先着3社限定オファー
12. **お申込みフォームセクション** - 入力フォーム + 送信ボタン
13. **追伸（P.S.）セクション** - 最後のひと押しメッセージ
14. **フッターセクション** - ロゴ + 会社情報 + コピーライト
